# Portfólio

Portfólio pessoal de Nicollas Trovão, desenvolvido com HTML, CSS e JavaScript puro.

## Estrutura

- `index.html`: estrutura e conteúdo da página.
- `css/styles.css`: estilos, responsividade e animações.
- `js/main.js`: comportamento da navegação e animações de entrada.
- `assets/images/`: pasta reservada para fotos, imagens de projetos e outros arquivos visuais.
- `assets/icons/`: ícones do site, incluindo a favicon.

## Como visualizar

Abra o arquivo `index.html` no navegador.
